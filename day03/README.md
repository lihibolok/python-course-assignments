# Day 03 – Nernst Potential Calculator

This program calculates the Nernst equilibrium potential (in mV) for an ion  
based on its intra- and extracellular concentrations, valence (z), and temperature.

## Files
- `nernst_logic.py` – core computation ("business logic")
- `cli.py` – command-line interface
- `tests/test_nernst_logic.py` – unit tests
- `requirements.txt` – dependencies

## Install
```bash
pip install -r requirements.txt
